# Habesha Connect V6.4.1

Shipping approval workflow update:
- Traveler accept does not reduce space.
- Sender payment goes to Payment Pending Admin Review.
- Admin approval reduces remaining trip space and reveals contact info.
- Delivered/Completed shipments show only in Admin.


## V6.4.1
- Removed the old manual Create Shipment form.
- Sender requests now happen only from traveler trip cards.
- Request Space now asks for sender name, sender phone, package description, receiver name, and receiver phone.
- Space is still reserved only after sender payment and admin approval.
